# Ampliwork Design System

## Core Identities
- **Background**: Midnight Navy (`#020617`) - Deep, premium, technical feel.
- **Accent**: Cyan (`#00A3FF`) - Electric, modern, high-contrast against navy.
- **Shapes**: Pill-shaped buttons (`rounded-full`) - Friendly yet modern.

## Architecture Guidelines
- **Typography**: Sans-serif, clean, high readability.
- **Spacing**: Generous, emphasizing "premium" feel.
- **Interaction**: Subtle hover states, smooth transitions (`duration-200`, `ease-in-out`).

## Component Specifications

### Buttons
- **Primary**: Cyan background (`bg-[#00A3FF]`), Midnight Navy text (`text-[#020617]`), pill-shaped (`rounded-full`), `font-semibold`. Hover: slightly lighter cyan or shadow loop.
- **Secondary**: Transparent with Cyan border (`border border-[#00A3FF]`), Cyan text (`text-[#00A3FF]`), pill-shaped.
- **Ghost**: Transparent, White/Gray text, hover: midnight navy lighter (`hover:bg-slate-800`).

### Cards / Surfaces
- **Background**: Very dark slate/navy, slightly lighter than main background (e.g., `#0f172a` or `bg-slate-900/50` with backdrop blur).
- **Borders**: Thin, subtle cyan or slate borders (`border-slate-800`).
- **Shadows**: Cyan glow for active/high-priority elements (`shadow-[0_0_15px_rgba(0,163,255,0.3)]`).

### Typography
- **Headings**: White (`text-white`), bold, tight tracking.
- **Body**: Slate-300 or Slate-400 (`text-slate-300`).
